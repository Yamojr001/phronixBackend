# Phronix AI - Laravel Backend (Standalone)

## Requirements
- PHP 8.2+
- Composer
- SQLite (included in PHP) or MySQL/PostgreSQL

## Quick Start

```bash
cd artifacts/laravel-api

# Run the setup script (handles everything automatically)
bash setup.sh

# Start the server
php artisan serve --host=0.0.0.0 --port=8000
```

## Manual Setup

```bash
# 1. Install dependencies
composer install

# 2. Configure environment
cp .env.example .env
php artisan key:generate

# 3. Edit .env and set required values:
#    GEMINI_API_KEY=your_key_here  (from https://aistudio.google.com/app/apikey)
#    DB_CONNECTION=sqlite           (or mysql/postgres)

# 4. Create database & run migrations
touch database/database.sqlite
php artisan migrate

# 5. Start server
php artisan serve --host=0.0.0.0 --port=8000
```

## Required Environment Variables

| Variable | Description |
|----------|-------------|
| `APP_KEY` | Auto-generated by `php artisan key:generate` |
| `GEMINI_API_KEY` | Google Gemini API key — powers all AI features |
| `DB_CONNECTION` | `sqlite` (default), `mysql`, or `pgsql` |

## Optional Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `GEMINI_DAILY_REQUEST_LIMIT` | `500` | Max Gemini API calls per day |
| `GEMINI_FALLBACK_MODEL` | `gemini-2.5-pro` | Fallback AI model |
| `MAIL_MAILER` | `log` | Mail driver (log = no real emails) |
| `GOOGLE_CLIENT_ID` | — | For Google OAuth login |
| `MICROSOFT_CLIENT_ID` | — | For Microsoft OAuth login |

## API Endpoints

All routes are prefixed with `/api`. Auth uses Sanctum Bearer tokens.

- `POST /api/register` — Create account
- `POST /api/login` — Get Bearer token
- `GET /api/dashboard` — Student dashboard
- `POST /api/tutor/explain` — AI tutor (Gemini)
- `POST /api/study-room/explain` — Study room AI
- `POST /api/reading-plans` — Generate AI reading plan
- `GET /api/courses` — List courses
